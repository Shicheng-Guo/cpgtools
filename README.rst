Install CpGtools using pip3 
----------------------------
::

 $ pip3 install cpgtools
 or 
 $ pip3 install git+https://github.com/liguowang/cpgtools.git
 or 
 $ pip3 install cpgtools-VERSION.tar.gz
 
Upgrade CpGtools
-----------------
::

 $ pip3 install cpgtools --upgrade
